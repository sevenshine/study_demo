###方式一：命令行
* 初始化
    * git init
* 将本地代码提交到本地仓库中
    * git add 
    * git commit -m '提交内容的描述'
    
* 将本地仓库与远程仓库进行绑定，并且push   
    * git remote add origin https://github.com/gujunling/test.git 
    * git push origin master （提交到github上）
